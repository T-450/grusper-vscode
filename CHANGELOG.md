# Change Log

All notable changes to the "grusper-theme" extension will be documented in this file.

## [1.1.0]
- Updated keyword font weight styling for improved readability.

## [1.0.0]
- Initial release of the Grusper theme.
- Added high-contrast dark background (`#1A1917`).
- Added semantic highlighting for keywords, functions, strings, and constants.
- Added custom UI colors for editor, terminal, and sidebar.
